Design a premium, ultra-minimalist mobile app UI for a car management application called "CarSync".

Design direction:
- Inspired by Tesla app, Apple (iOS), and high-end automotive dashboards
- Clean, minimal, but with strong visual hierarchy and contrast
- Premium and modern feel, not flat or lifeless
- Use depth subtly through contrast and color

Color system:
- Primary background: #FFFFFF
- Secondary background: #F7F7F7
- Divider / subtle borders: #EAEAEA

Primary text:
- #1C1C1E

Secondary text:
- #9E9E9E

Accent color (main interactive color):
- #3B82F6 (soft blue, used sparingly)

Secondary premium color (NEW — for depth and hierarchy):
- Navy blue / dark gray: #1F2937
- Used for:
  - Important cards (highlight sections)
  - Headers or top areas
  - Selected states
  - Key emphasis elements

Status colors:
- Success: #22C55E
- Warning: #F59E0B

Typography:
- Clean sans-serif (Inter or SF Pro style)
- Large bold headings
- Medium-weight key data
- Light secondary labels
- Strong hierarchy

Layout & components:
- Card-based layout with rounded corners (12–16px)
- Soft shadows (very subtle)
- Generous whitespace
- Minimal outline icons
- Avoid heavy borders

Visual balance:
- Mix white cards with occasional dark (navy) cards for contrast
- Use dark sections to highlight important information (e.g., car status)
- Maintain overall minimalism while adding visual depth

Create these screens:

1. Home (Dashboard):
- Header with car name ("My Car")
- Main premium card (use navy/dark background):
  - Car health score ("92% Healthy")
  - Next maintenance ("Oil change in 12 days")
- Below:
  - Fuel level (progress bar)
  - Monthly expenses (large number)
  - Mileage
  - Maintenance
- Use contrast between white and dark cards

2. Expenses:
- Large monthly total
- Minimal breakdown
- Clean list or subtle chart
- Optional dark highlight card for total

3. Maintenance:
- Checklist layout
- Items with status indicators
- Clean spacing and icons

4. Alerts:
- Minimal notification list
- Clear hierarchy and readability

5. Profile:
- User info
- Car info
- Settings
- Clean structure

Navigation:
- Bottom navigation bar
- Minimal icons
- Active state can use navy or accent blue

Micro-interactions:
- Smooth transitions
- Subtle press/hover states
- Premium feel

Style constraints:
- Maintain minimalism
- Avoid clutter and heavy effects
- Use dark color strategically, not excessively
- Focus on clarity, elegance, and usability

Goal:
Create a Tesla-like experience with Apple-level refinement, combining minimalism with depth using a strong secondary dark color.
